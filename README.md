# Clash Royale Match Predictor

## プロジェクト概要
- クラッシュロワイヤル（Clash Royale）の8枚vs8枚のデッキ情報から、対戦の勝敗(相性)を予測する深層学習モデルのAPIです。
- URL：https://clash-royale-api-824518372257.asia-northeast1.run.app/docs

### プロジェクトの・動機・目的
- クラロワのデッキ同士の相性はそれぞれの主観に大きく依存し、争いの火種になることがあります。そこで、２つのデッキを入力とし過去の大規模データから勝敗（相性）を予測することで、客観的なデッキ相性の提案をすることを目的とします。
- クラロワの勝敗は「個々のカードの強さ」ではなく、「カード同士のシナジー（共起）」と「メタゲーム（有利不利のじゃんけん）」によって決まります。
この「非対称で複雑な関係性」をAIにどう理解させるか？という課題に対して、アテンション機構やGNNによるグラフ構造を用いたアプローチで挑みます。

## 既存実験
クラロワにおける機械学習を用いた勝敗予測は、主に「デッキ構成（相性）」が勝敗に与える寄与度を定量化する試みとして行われてきました。主要な先行実験の結果を以下にまとめる。

1. デッキ構成による予測限界の検証 (John Leonardo, 2024)
70,200件のリアルタイムバトルデータをAPI経由で取得し、プレイヤーのスキルやレベル、タワーHPを排除した「8枚のカード構成のみ」での予測精度を検証したリサーチ。
- データセット：70,200件の対戦ログ
- 特徴量：474次元のバイナリベクトル（237種類のカード × プレイヤー2人）
- 予測精度：
  - LightGBM: 56.94%
  - ニューラルネットワーク: 57.25%
- 知見: 50.0%（ランダム）を有意に上回ることから、デッキ構成自体に確実な予測力があることが証明されたが、同時に勝敗の約4割以上はスキルやタイミング、運などの「デッキ外要因」に依存していることが示唆された。

2. 高度な特徴量とモデルの最適化 (Joel Foo, "Clash-ML")
デッキ構成に加えて「カードレベル」の情報を特徴量に統合し、ニューラルネットワークのチューニングを行ったプロジェクト。
- 手法: 多層パーセプトロン (MLP) およびロジスティック回帰
- 予測精度:
  - ロジスティック回帰: 63.9%
  - MLP (最適化後): 67.7%
- 知見: プレイヤーの経験値を考慮しない場合、デッキとレベルの情報のみでは 67.7% が予測精度の限界値（これ以上の向上は過学習を招く）であると報告された。

3. 大規模コンペティション (FedCSIS 2019 Clash Royale Challenge)
1億6,000万回以上の対戦に基づく大規模データセットを用いたデータマイニングコンペティションの結果。
- 手法: パラレル化された「Greedy Search」と サポートベクター回帰 (SVR) を組み合わせた手法
- 成果: データセット全体のわずか1%を利用するだけで、高精度な勝率予測モデルの構築が可能であることが示され、効率的なデータ編集戦略の有効性が確認されました。

## 基本アーキテクチャ
- 入力：デッキのカード8枚（自分、相手の計16枚）
- 出力：勝率()

## 使用データ
- 公式API`https://developer.clashroyale.com/#/`から得られたデータを元に勝率を予測。
- 収集したデッキや対戦データはある程度の実力の拮抗を保証するため、グローバルランキング上位のプレイヤーを対象にして対戦履歴から芋づる的に収集した。
    - デッキデータ：60万件
    - 対戦データ：5000件（実力者同士の対戦結果）
